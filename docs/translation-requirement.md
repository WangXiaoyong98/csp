# CSP项目翻译规范

## IP外设配置文件翻译

### 文件位置

`resources/database/ip/peripherals/` 目录下的外设配置文件

### 翻译原则

- 所有 zh-cn 字段需要翻译成简洁、准确的中文名称
- 描述性文字要流畅自然，符合技术文档的表达习惯
- 保持专业术语的准确性
- 英文对照字段（en）保持不变

### 特殊标记保留

以下内容**不要翻译**：

- 分频值：`/1`、`/2`、`/4`、`/8` 等
- 数学表达式：`4096 x 2**WDGTB`、`2^WDGTB` 等
- 十六进制：`0x40`、`0x3F` 等
- 公式：`(PCLK/4096)/8` 等
- 变量名、寄存器名：`CNT6`、`PCLK` 等

### 格式要求

- 保持原文件格式（YAML/JSON/MD等）不变
- 注意缩进，确保格式正确
