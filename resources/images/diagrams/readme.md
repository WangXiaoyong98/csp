# 文件说明

本文件夹内包含多个 **SVG 矢量图文件**，这些文件均为 **可编辑格式**，可根据需要进行二次修改或重新生成。

部分文件来自手动设计（可用 **Draw.io** 编辑），部分由 **PlantUML** 自动生成

## 🎨主题适配规范

为支持软件主题切换（亮/暗模式），所有 SVG 均需满足以下要求：

1. 背景必须保持透明（不要填充白色或其他背景色）。
2. 黑色 (#000000) 元素会在运行时自动替换为软件主题色。
3. 白色 (#FFFFFF) 元素会在运行时自动替换为软件背景色。
4. 若需固定颜色请使用非纯黑或纯白色。
